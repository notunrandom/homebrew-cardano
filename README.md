# Notunrandom Cardano

## How do I install these formulae?

`brew install notunrandom/cardano/<formula>`

Or `brew tap notunrandom/cardano` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "notunrandom/cardano"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
