# Homebrew Cardano

This is a Homebrew tap to install the Cardano node and some tools.

## How do I install these formulae?

`brew install notunrandom/cardano/<formula>`

Or `brew tap notunrandom/cardano` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "notunrandom/cardano"
brew "<formula>"
```

## Documentation

List of formulae:

- `cardano-node`: the Cardano node (10.5.1) and CLI
